# My NixOS Configuration

NixOS flake configuration for **guillermo** — a KDE Plasma 6 desktop
running in the `America/Phoenix` timezone.

## Quick Start

Activate your changes:

```sh
sudo nixos-rebuild switch --flake .#guillermo
```

## Structure

```
flake.nix          ← Flake entry point, loads modules
├── nixos/
│   ├── configuration.nix      ← Main system config (bootloader, network, audio, etc.)
│   └── hardware-configuration.nix  ← Auto-generated by nixos-generate-config
└── modules/
    ├── desktop.nix            ← KDE Plasma 6, SDDM, US keymap, user account
    ├── programs.nix           ← Shell programs, git config, system packages
    ├── llm.nix                ← Ollama + Open WebUI (commented out by default)
    └── minecraft.nix          ← Minecraft server (commented out by default)
```

## System Overview

| Feature | Details |
|---|---|
| **Desktop** | KDE Plasma 6 with SDDM login manager |
| **Audio** | PipeWire (PulseAudio compatibility enabled) |
| **Network** | NetworkManager, SSH daemon |
| **Print** | CUPS enabled |
| **Keyboard** | US layout |
| **Timezone** | America/Phoenix |
| **Locale** | en_US.UTF-8 |
| **Boot** | systemd-boot (UEFI) |
| **Never sleep** | Sleep/suspend/hibernate disabled (always-on machine) |

## Optional Modules

### LLM Services (llm.nix)

Local AI inference with Ollama and a web UI.

- **Ollama** — GPU-accelerated via Vulkan, loads `gemma4:e2b` and `qwen3.6:35b` on startup
- **Open WebUI** — web-based chat interface (port 8080)

To enable, add `./modules/llm.nix` to the `modules` list in `flake.nix`.

### Minecraft Server (minecraft.nix)

Fully declarative Minecraft Java Edition server in survival mode with a whitelist.

To enable, add `./modules/minecraft.nix` to the `modules` list in `flake.nix`.

## Notes

- **Hardware config** (`nixos/hardware-configuration.nix`) is auto-generated and should
  not be edited manually. Regenerate it with `nixos-generate-config` if hardware changes.
- **VS Code remote SSH** works via `programs.nix-ld.enable = true`.
